Pour faire fonctionner le programme :

1. Créez une base de données SQL.
2. Entrez les identifiants de connexions à votre base de données dans le fichier "/include/connexionBDD.php"
3. Exécutez le fichier "/db/gestion_academique.sql"
4. Profitez de l'application. Plus d'informations sur le rapport de projet.
